            <li>Use SelectableTag for checkbox-like selection.</li>
            <li>Use DismissibleTag for a removable value.</li>
            <li>Use OperationalTag for a compact action or link.</li>
          </ul>
          <h3>When not to use</h3>
          <ul>
            <li>Do not use tags as primary page actions.</li>
            <li>Do not use a read-only tag when the element changes state or navigates.</li>
